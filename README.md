Pinoccio, Johnny-Five and your Mac OSX Mouse
============================================

Control your Mac OSX Mouse Cursor, using the power Johnny-Five, Pinoccio and Node.js

###Parts Needed
- [Pinoccio](http://pinocc.io)
- 2 Push Buttons
- Jumper Wires
- 2-Axis Joystick (I got mine at Radioshack for $10)
- Breadboard
